# Project Images Setup

## Required Images

Place project photos in `public/images/projects/` with these filenames:

1. **villa-mauritius.jpg** - Luxury villa with solar panels
2. **hotel-cayman.jpg** - Boutique hotel installation
3. **factory-germany.jpg** - Manufacturing facility
4. **farm-mauritius.jpg** - Agricultural processing facility
5. **residence-germany.jpg** - Smart home installation
6. **resort-cayman.jpg** - Eco-resort system

## Image Specifications

- **Format**: JPG or PNG
- **Dimensions**: 1200x900px (4:3 aspect ratio)
- **File size**: < 500KB each
- **Style**: Professional installation photos showing:
  - Solar panels on roofs
  - Battery systems in place
  - Wide shots of buildings
  - Before/after comparisons (optional)

## Quick Setup

```bash
public/images/projects/
├── villa-mauritius.jpg
├── hotel-cayman.jpg
├── factory-germany.jpg
├── farm-mauritius.jpg
├── residence-germany.jpg
└── resort-cayman.jpg
```

## Photo Tips

- Show completed installations
- Include building context
- Good lighting (daytime)
- Professional composition
- Clean, uncluttered backgrounds

## Temporary Placeholders

Until you add real photos, use:
```typescript
image: 'https://placehold.co/1200x900/1a1f1b/00c853?text=Project+Name'
```

Replace with real installation photos before launch.
